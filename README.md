# Deep Learning for Computer Vision

This repository is part of a Computer Vision course at HCMUS. The goal is to build and train deep learning models from scratch for various computer vision tasks, and evaluate their performance on standard benchmarks.

---

## 🚀 Setup Environment

### 1. Create a virtual environment

```bash
python -m venv venv
```

### 2. Activate the environment

* **Linux/macOS**

```bash
source venv/bin/activate
```

* **Windows (Command Prompt)**

```bash
venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install --upgrade pip
pip install -r requirements.txt
```
